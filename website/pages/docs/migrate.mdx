# Swagger to NestJS
## Outline
```bash
npm install -g @nestia/migrate
npx @nestia/migrate swagger.json output_directory
```

`@nestia/migrate` can convert `swagger.json` file to a NestJS project.

Also, as `@nestia/sdk` can generate those things, every backend languages and frameworks also can do those things just by utilizing `@nestia/migrate`. Let's enjoy the new world.
